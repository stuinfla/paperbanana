About PaperBanana SVG

PaperBanana is a multi-agent framework for generating publication-quality diagrams from text descriptions. Originally developed by Peking University and Google Cloud AI Research for academic paper figures, this fork adds a storytelling-enhanced SVG pipeline that produces vector diagrams with 100% text accuracy.

How it works: A Planner agent finds a visual metaphor ("What is this LIKE?"), a Stylist agent applies NeurIPS-level aesthetic guidelines, and an SVG Visualizer generates clean vector code rendered by Cairo. A Critic agent can evaluate the rendered output and trigger code fixes.

Key advantages over raster image generation:

• 100% text accuracy — text is rendered by Cairo, not predicted by a neural net
• Infinitely scalable — vector output works at any resolution
• Fully editable — output is SVG code, version-controllable and diffable
• 10x faster — ~30 seconds per diagram vs 3-5 minutes for raster
• 20x cheaper — ~$0.05 per diagram vs $0.50-2.00 for multi-agent raster

Models: gemini-3.1-pro-preview (planning/styling/SVG generation) + cairosvg (rendering)
